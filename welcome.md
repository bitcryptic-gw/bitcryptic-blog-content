This blog is where we write about building and running crypto-native infrastructure — self-hosted AI agents, DePIN compute and other networks, Bitcoin nodes, and the tools and patterns that make it all work among other random musings.

Posts here are written in markdown, version-controlled in git, and served via a self-hosted Astro instance on our own hardware. No WordPress, no cloud lock-in.

Topics you'll find here:

- **AI infrastructure** — local LLMs, agent platforms, agentic payment rails (x402)
- **DePIN** — decentralised physical infrastructure networks, node operation, economics
- **Self-hosting** — Docker, Unraid, Tailscale, Caddy, and the tools that make it practical
- **Bitcoin/crypto** — node operation, solo mining, on-chain tooling
- **Networking** — Ubiquiti, mesh networking, home automation
